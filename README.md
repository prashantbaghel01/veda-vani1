# Veda Vani

A book summary web app inspired by Indian scriptures like Vedas, Upanishads, Ramayana, and Mahabharata.

## Features
- Library of books
- Book reader with page navigation
- Search functionality
- Clean reading UI

## Tech Stack
- React (Vite)
- Node.js
- MongoDB
